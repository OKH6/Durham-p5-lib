This sketch draws an image of 2D trees that are coloured in such a way that they look 3D. At the top of the page you will find controls to change various properties of the sketch, they are as such:

  • Free tree mode: Enabling this means that when the left mouse button is clicked a new tree will be grown at the current cursor position on the sketch. This cannot be enabled at the same time as the 3D texture example.
  • 3D texture example: Enabling this renders the sketch on each face of a spinning 3D cube. This cannot be enabled with Free tree mode.
  • Number of columns: This slider allows you to choose the number of columns of trees to be displayed. The range is between 0 and 10.
  • Number of rows: This slider allows you to choose the number of rows of trees to be displayed. The range is between 0 and 10.
  • Maximum lifespan of branch: This slider allows you to choose the time it takes for a branch to split.
  • Random tree mode: Enabling this means that the sketch renders a number of trees at random locations on the screen.
  • Number of trees for random mode: This slider allows you to choose the number of trees to be randomly rendered when Random tree mode is enabled.
  • submit: Pressing this button refreshes the sketch with the variables selected by the user.
  • Tutorial: Pressing this button displays this screen.
  • Sketch: Pressing this button displays the sketch instead of this page.
